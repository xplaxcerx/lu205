# Инструкция по настройке проекта в Wokwi

## Способ 1: Добавление компонентов вручную

1. Откройте [wokwi.com](https://wokwi.com)
2. Создайте новый проект Arduino
3. Скопируйте код из `arduino_project.ino`

## Добавление компонентов:

### 1. Arduino UNO
- Уже добавлен по умолчанию

### 2. DHT22
- Нажмите кнопку **"+"** (Add Part)
- Найдите **"DHT22"**
- Добавьте компонент
- Подключите:
  - **VCC** → **5V** Arduino
  - **GND** → **GND** Arduino  
  - **OUT** → **Pin 2** Arduino

### 3. MAX7219 с LED матрицей (4 модуля)
- Нажмите кнопку **"+"** (Add Part)
- Найдите **"MAX7219"** или **"LED Matrix 8x8"**
- Добавьте **4 модуля** (для матрицы 8x32)

#### Подключение первого MAX7219:
- **VCC** → **5V** Arduino
- **GND** → **GND** Arduino
- **DIN** → **Pin 11** Arduino
- **CS** → **Pin 10** Arduino
- **CLK** → **Pin 13** Arduino

#### Каскадное подключение остальных модулей:
- **DOUT первого** → **DIN второго**
- **DOUT второго** → **DIN третьего**
- **DOUT третьего** → **DIN четвертого**
- **CS всех модулей** → **Pin 10** Arduino (общий)
- **CLK всех модулей** → **Pin 13** Arduino (общий)
- **VCC и GND всех модулей** → **5V и GND** Arduino

## Альтернативный способ: Использование готового шаблона

1. В Wokwi найдите примеры проектов
2. Найдите проект с MAX7219 и LED матрицей
3. Скопируйте схему оттуда
4. Добавьте DHT22 по инструкции выше
